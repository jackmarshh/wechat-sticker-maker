# 表情包生成提示词模板

使用时将 `[CHARACTER]`、`[STYLE]`、`[SCENE]` 替换为实际内容。

---

## 宫格图（主要表情）

### 16 张 — 4×4 宫格

```
A clean 4x4 grid of 16 unique [CHARACTER] cartoon sticker expressions,
pure white background, thick black outline, [STYLE] style,
consistent character design throughout all cells,
equal cell spacing with subtle grid lines visible,
16 different emotions:
happy, sad, angry, surprised, winking, sleeping, thinking, eating,
running, dancing, crying, laughing, cool with sunglasses, waving, heart eyes, thumbs up.
Each sticker clearly separated, no overlapping between cells.
```

### 24 张 — 3×8 宫格

```
A clean 3x8 grid (3 rows and 8 columns) of 24 unique [CHARACTER] cartoon sticker expressions,
pure white background, thick black outline, [STYLE] style,
consistent character design throughout all cells,
equal cell spacing with subtle grid lines visible,
24 different emotions:
happy, sad, angry, surprised, winking, sleeping, thinking, eating,
running, dancing, crying, laughing, cool with sunglasses, waving, heart eyes, thumbs up,
shy blushing, excited jumping, bored yawning, confused question mark,
celebrating with confetti, sick with thermometer, working at desk, friendly greeting wave.
Each sticker clearly separated, no overlapping between cells.
```

## 辅助素材

### 详情页横幅 (750×400)

```
[CHARACTER] [STYLE] in a [SCENE], wide banner illustration,
750x400 pixels aspect ratio (wider than tall), no text whatsoever,
vibrant colorful background scene, promotional banner composition,
character featured prominently, eye-catching colors
```

**场景建议 [SCENE]：**
- 咖啡馆：`cozy coffee shop interior`
- 公园野餐：`sunny park picnic scene`
- 星空夜晚：`magical starry night sky`
- 城市街道：`colorful city street`
- 居家场景：`cute home living room`

### 赞赏引导图 (750×560)

```
[CHARACTER] [STYLE] warm appreciation illustration,
750x560 pixels aspect ratio, soft warm pastel colors,
heartwarming cozy scene suggesting generosity and gratitude,
no text overlay, gentle inviting atmosphere
```

### 赞赏致谢图 (120×120)

```
[CHARACTER] [STYLE] grateful thank you expression,
120x120 pixels, pure white background, thick black outline,
simple clean design, character bowing or waving with gratitude,
high contrast for small display size
```

## 角色描述建议

描述角色时，指定以下要素效果更好：

```
[SPECIES/TYPE] [DISTINGUISHING COLOR] [SIZE] [KEY FEATURE],
e.g.:
- "a chubby orange tabby cat with big round eyes"
- "a small white fluffy bunny with long floppy ears and a pink bow"
- "a round green frog wearing a tiny yellow raincoat"
- "a baby panda with large black eye patches"
```
