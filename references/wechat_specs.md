# 微信静态表情包制作规范（2024/2025）

> 来源：微信表情开放平台官方文档，最后更新：2025-05

---

## 1. 表情主图 (Main Stickers)

| 参数 | 规格 |
|------|------|
| 数量 | **16 张**或 **24 张**（只能选其一）|
| 格式 | PNG，**必须透明背景** |
| 尺寸 | **240 × 240 px**，精确匹配 |
| 文件大小 | 每张 ≤ 500 KB |
| 命名建议 | `sticker_01.png` ~ `sticker_16.png` |
| 内容要求 | 角色居中，必须有白色或彩色描边以区分背景 |

---

## 2. 表情缩略图 (Thumbnails)

| 参数 | 规格 |
|------|------|
| 数量 | 与主图一致（16 或 24 张）|
| 格式 | PNG，**必须透明背景** |
| 尺寸 | **120 × 120 px** |
| 文件大小 | 每张 ≤ 200 KB |
| 命名建议 | `thumb_01.png` ~ `thumb_16.png` |

---

## 3. 表情封面图 (Cover Image)

| 参数 | 规格 |
|------|------|
| 数量 | 1 张 |
| 格式 | PNG，**必须透明背景** |
| 尺寸 | **240 × 240 px** |
| 内容要求 | 选取最具代表性的一张表情作为封面 |
| 命名 | `cover.png` |

---

## 4. 聊天面板图标 (Chat Panel Icon)

| 参数 | 规格 |
|------|------|
| 数量 | 1 张 |
| 格式 | PNG，**必须透明背景** |
| 尺寸 | **50 × 50 px** |
| 内容要求 | 角色标志性形象，小尺寸下仍清晰可辨 |
| 命名 | `panel_icon.png` |

---

## 5. 详情页横幅 (Detail Page Banner)

| 参数 | 规格 |
|------|------|
| 数量 | 1 张 |
| 格式 | PNG 或 JPG |
| 尺寸 | **750 × 400 px** |
| 文件大小 | ≤ 1 MB |
| 内容要求 | 展示表情主题和角色，背景丰富，**不得有文字** |
| 命名 | `banner.png` / `banner.jpg` |

---

## 6. 赞赏素材 (Appreciation Assets)

### 赞赏引导图

| 参数 | 规格 |
|------|------|
| 数量 | 1 张 |
| 格式 | PNG 或 JPG |
| 尺寸 | **750 × 560 px** |
| 内容要求 | 温馨插画，引导用户打赏，与主题风格统一 |
| 命名 | `appreciation_guide.png` |

### 赞赏致谢图

| 参数 | 规格 |
|------|------|
| 数量 | 1 张 |
| 格式 | PNG |
| 尺寸 | **120 × 120 px** |
| 内容要求 | 角色感谢表情 |
| 命名 | `appreciation_thanks.png` |

### 赞赏引导语

- **字数**：5 ~ 15 个字
- **语气**：轻松亲切，鼓励打赏
- 保存于 `appreciation_text.txt`

---

## 7. 内容合规要求

- **不得含有**：政治敏感内容、违禁符号、版权素材（含 IP 形象未授权使用）
- **不得含有**：广告信息、联系方式、外部链接
- **角色设计**：必须为原创，不得仿冒已上架表情包角色
- **背景**：主图和封面必须为透明背景（PNG alpha 通道）
- **描边**：建议白色或黑色描边，确保在深色/浅色聊天背景下均清晰可见

---

## 8. 提交资料清单

提交至微信表情开放平台时，还需准备：
- 表情名称（中文，5-10 字）
- 表情简介（30 字以内）
- 标签（3-5 个关键词）
- 作者名称和简介
- 赞赏引导语（如启用赞赏功能）

---

## 9. 审核周期参考

| 阶段 | 时间 |
|------|------|
| 初审 | 1-3 个工作日 |
| 修改后重审 | 1-2 个工作日 |
| 上线后推广 | 视平台推荐机制 |

常见审核不通过原因：
1. 素材尺寸不符（最常见）
2. 背景未透明化
3. 含文字（横幅、引导图）
4. 风格前后不一致
5. 角色与封面/横幅不一致
